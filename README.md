# чтобы запустить фронт

### Установить свежую версию node.js если нет
- https://nodejs.org/en/

### Установить или обновить npm:
- npm install -g npm

### Установить angular-cli:
- npm install -g @angular/cli

### перейти в .../organizer_frontend 
- npm run start
- 
### в браузере
- http://localhost:4200/
